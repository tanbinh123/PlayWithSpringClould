# Zuul

Zuul 的主要功能是路由转发和过滤器。路由功能是微服务的一部分，比如 `/api/user` 转发到到 user 服务，`/api/shop` 转发到到 shop 服务。
zuul 默认和 Ribbon 结合实现了负载均衡的功能。

zuul 有以下功能：

- Authentication
- Insights
- Stress Testing
- Canary Testing
- Dynamic Routing
- Service Migration
- Load Shedding
- Security
- Static Response handling
- Active/Active traffic management
